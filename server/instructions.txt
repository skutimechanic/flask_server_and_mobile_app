utworzenie wirtualnego środowiska: python -m venv venv

uruchomienie z konsoli wirtualnego środowiska: w folderze server odpalić konsole i wpisac komende venv\Scripts\activate.bat

w wirtualnym środowisku zainstalować flaska: pip install flask
w wirtualnym środowsiku zainstalować: pip install python-dotenv

odpalenie wirtualnego środowiska w vs code: https://stackoverflow.com/questions/54106071/how-can-i-set-up-a-virtual-environment-for-python-in-visual-studio-code
